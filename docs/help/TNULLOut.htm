TNULLOut

Properties Methods Events

Unit: ACS_Audio

Descends from: TACSOutput

TNULLOut component is different from all other output components in that it generates no output. This component is designed in order to be the end of the sound processing chain for situations when you don't need the actual output (for example, when you just want to perform some analisys of the incoming signal. This component doesn't introduce any new properties or methods.
