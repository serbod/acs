TMP3ToWav Methods

Run;

Call this method to start the converter. After Run is called Buisy property returns True, indicating the that component cannot perform other jobs. Running TMP3ToWav component generates OnProgress events to indocate its progress, and OnDone event when the current job is finished.

Stop;

Call this method to stop the conversion process prematurely. OnDone event will be generated when this method is called.
