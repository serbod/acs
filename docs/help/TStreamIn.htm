TStreamIn

Properties Methods Events

Unit: ACS_Streams

Descends from: TACSStreamedInput

TStreamIn component can acquire raw audio data from any TStream descendant assigned to its Stream property. Since data is not formatted this input component cannot tell such parameters as the sample rate or the number of channels. You should assign these explicitly via the component's corresponding properties.

See also: TStreamOut
