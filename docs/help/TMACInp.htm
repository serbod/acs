TMACIn Properties

AverageBitrate : Integer;

Read Only

This property shows the average bitrate for the playing ape file.

CurrentBitrate : Integer;

Read Only

This property shows the current bitrate for the playing ape file.

CurrentBlock : Integer;

Read Only

This property shows the number of the block being read from the file.

CurrentMS : Integer;

Read Only

This property shows the number of milliseconds from the beginning of the file being read.

LengthMS : Integer;

Read Only

This property shows the length of the file in milliseconds.

TotalBlocks : Integer;

Read Only

The total number of blocks in the file.
