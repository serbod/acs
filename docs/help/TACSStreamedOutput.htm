TACSStreamedOutput

Properties Methods Events

Unit: ACS_Classes

Descends from: TACSOutput

TACSStreamedOutput introduces Stream property. This property allows output components that descend from TACSStreamedOutput to store data not only to files on disc but to any kind of stream as well. The descendats of this class are TMP3Out, TVorbisOut, TWaveOut and TStreamOut components.
