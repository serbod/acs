TACSConverter

Properties Methods Events

Unit: ACS_Classes

Descends from: TACSInput

TACSConverter is the base class for all ACS converter classes. Converters are intermediate sections in the audio processing chain. Being the descendants of TACSInput they can be linked to the output components, while their Input property allows to link them with input components and other converters.

TACSConverter introduces new published property - Input - the very property that distinguishes a converter from an input component.
