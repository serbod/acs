TCDIn

Properties Methods Events

Unit: ACS_CDROM

Descends from: TACSInput

TCDIn component is an input component that allows to read the data from audio CD tracks in a way CD-rippers do. In order to pass audio data from TCDIn to some output or converter component, you have to assign CDIn component to output or converter component's Input property and specify starting and ending positions for data transfer.

Windows note: under Windows TCDIn component requires CDRip.dll

See also: TCDPlayer
