TRateConverter Properties

FilterWindow : TFilterWindowType;

Use this property to set the type of the window smoothing the sinc filter frequency responce. The possible values of this property are fwBlackman, fwHamming, and fwHann.

KernelWidth : Integer;

Specifies the number of the sinc filter kernel points. Larger numbers provide better filter quality by the cost of higher CPU load or slower performance.

OutSampleRate : Integer;

Use this property to set the sample rate in Hz for the output stream.
