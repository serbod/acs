TMPEGIn

Properties Methods Events

Linux only

Unit: ACS_Misc

Descends from: TACSFileIn

TMPEGIn component reads audio data from MPEG compressed streams (including MP3). In order to work with the resulting audio stream you have to assign MPEGIn component to Input property of some converter or output component.

Note that using TMPEGIn requires libsmpeg.so shared library to be installed in your system.
