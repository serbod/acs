TCDPlayer

Properties Methods Events

Unit: ACS_CDROM

Descends from: TComponent

TCDPlayer component encapsulates CD player functionality, such as obtaining audio-disc info, positioning, playing, stopping, pausing audio-tracks and so on. TCDPlayer doesn't process the CD audio-data by itself. It just controls CD-ROM device/soundcard playing capabilities.

See also: TCDIn
