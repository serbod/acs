TACSStreamedInput

Properties Methods Events

Unit: ACS_Classes

Descends from: TACSInput

TACSStreamedInput introduces Stream property. This property allows input components that descend from TACSStreamedInput to get data not only from files on disc but from any kind of stream. The descendats of this class are TACSFileIn (and all its descendats except TMACIn), and TStreamIn component that is designed to read raw audio data from streams.

See also TACSStreamedOutput
