TSampleConverter

Properties Methods Events

Unit: ACS_Converters

Descends from: TACSConverter

TSampleConverter converts 8-bit unsigned input audio samples into 16-bit signed output and vice versa. Like all other converter components, TSAmpleConverter has an Input property so it can accept input from other input or converter components. As a descendant from TACSInput, TMSConverter can be itself a source of input for other components.

See also: TRateConverter, TMSConverter
